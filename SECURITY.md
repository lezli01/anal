# Security Policy

## Supported Versions

`anal` is pre-1.0.0. Security fixes are provided for the latest published
minor version.

| Version | Supported |
| ------- | --------- |
| 0.1.x   | Yes       |
| < 0.1   | No        |

## Reporting a Vulnerability

Please report suspected vulnerabilities through GitHub's private vulnerability
reporting flow for this repository when available. If that is not available,
open a GitHub issue with only non-sensitive details and ask for a private
maintainer contact path.

Include:

- affected versions or commits;
- steps to reproduce;
- potential impact;
- any known workaround.

Maintainers will acknowledge valid reports as quickly as practical and will
coordinate fixes before public disclosure.
